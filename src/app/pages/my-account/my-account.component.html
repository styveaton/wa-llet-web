Comment puis-je changer le nom sur mon compte ?

Le nom de votre compte Wallet doit être votre nom légal complet tel qu'il apparaît sur votre pièce d'identité officielle.
Si vous avez fait une erreur lors de l'inscription (par exemple, votre nom est mal orthographié), contactez-nous.

Comment puis-je changer la date de naissance sur mon compte ?

La date de naissance sur votre compte Wallet doit être la date de naissance exacte telle qu'elle apparaît sur votre pièce d'identité officielle.
Si vous avez fait une erreur lors de l'inscription (par exemple, votre date de naissance est incorrecte), contactez-nous.

Comment puis-je changer l'adresse et le numéro de téléphone sur mon compte ?

L'adresse sur votre compte Wallet doit être votre adresse résidentielle complète. Pour modifier votre numéro de téléphone ou votre adresse personnelle, connectez-vous sur notre appli ou notre site Web et accédez à la rubrique Mes coordonnées.
Si vous n'avez plus accès au numéro de téléphone enregistré, veuillez contacter le service client qui pourra vous aider. Vous trouverez nos coordonnées ici

Comment puis-je changer le pays d'envoi sur mon compte ?

Le « pays d'envoi » sur votre compte Wallet doit être le pays où vous vivez actuellement. Ne modifiez pas le pays d'envoi, sauf si vous avez déménagé de façon définitive dans un autre pays.
Si vous avez déménagé dans un autre pays ou commis une erreur lors de l'enregistrement (par exemple, vous avez choisi le mauvais pays), contactez nous.

Comment puis-je modifier l'adresse e-mail sur mon compte ?

Pour modifier l'adresse e-mail enregistrée, connectez-vous sur notre appli ou notre site Web et accédez à Mes coordonnées.
Si vous rencontrez des problèmes avec cette opération, contactez-nous.

Comment puis-je changer le mot de passe de mon compte ?

Si vous avez oublié votre mot de passe pour le site internet :
  1. Rendez-vous sur la page de connexion du site Wallet
  2. Cliquez sur "mot de passe oublié ?"
  3. Saisissez votre adresse e-mail et cliquez sur "Envoyer"
  4. Vérifiez votre boite mail, nous vous aurons envoyé un e-mail incluant un lien.
  5. Cliquez sur le lien et vous serez guidé dans le processus de configuration d'un nouveau mot de passe

Comment puis-je changer le code PIN de mon compte ?

Lorsque vous utilisez l'application Wallet sur Android et iOS, vous devrez déterminer un code PIN afin de sécuriser votre compte. Si vous avez oublié le code PIN :
  1. Ouvrez l'appli.
  2. Cliquez sur « Oublié ? ».
  3. Saisissez votre adresse e-mail et votre mot de passe.
  4. Cliquez sur « Se connecter ».
  5. Vous serez invité à sélectionner un nouveau code PIN.
Pour réinitialiser votre mot de passe, cliquez sur « Oublié ? » sur l'écran à l'étape 3.

Puis-je avoir plusieurs comptes Wallet ?

Nous n'autorisons qu'un seul compte par client.

Puis-je partager un compte Wallet ?

Nous n'autorisons pas les comptes joints ou plusieurs titulaires de compte à l'heure actuelle. Chaque client doit créer son propre compte.
En effet, nous devons nous assurer que nous pouvons contrôler l'identité de chaque personne qui utilise Wallet.

Comment fermer mon compte Wallet ?

Pour désactiver votre compte Wallet, contactez nous.
Si vous changez d'avis, vous pouvez toujours réactiver votre compte en nous contactant.

Comment supprimer l'historique de mes transactions Wallet ?

Wallet est légalement tenu de tenir conserver l'historique des transactions de nos clients. Cela signifie que nous ne pouvons pas supprimer de transactions.

Comment puis-je déposer une plainte ?

Si vous n'êtes pas satisfait du service que vous avez reçu de Wallet, contactez-nous. Nous vous répondrons dès que possible, merci de patienter.
Nous espérons rectifier la situation de manière satisfaisante. Mais si vous souhaitez appuyer votre plainte après avoir reçu notre réponse ou offre écrite, veuillez vous référer à la section « Réclamations » de nos Conditions générales. Vous y trouverez le processus de plainte relatif à la juridiction dans laquelle vous vivez.

Plaintes depuis le Royaume-Uni, l'UE et les États-Unis

Si vous vivez au Royaume-Uni et n'êtes pas satisfait de notre réponse, vous pouvez la transmettre au FOS (Financial Ombudsman Service).
Si vous vivez dans l'Union européenne, vous pouvez trouver des options pour résoudre votre réclamation sur la plateforme de résolution des litiges en ligne (ODR).
On peut vous demander un numéro de référence ou des détails sur votre correspondance avec Wallet. Vous pouvez utiliser dataprivacy@Wallet.com comme adresse électronique.
Si vous vivez aux États-Unis, accédez à la page Déposer une plainte - USA.
